# PA 壶喷芯网页版工具说明

## 这次继续做了什么

我把前面那版命令行脚本，扩成了一个可直接调参数的本地网页工具：

- 文件：`foam_cannon_tool.html`
- 作用：尽量简化输入地给出喷芯推荐区间和标准喷芯建议
- 额外能力：
  - 明确输入逻辑：`流量 + 压力` 或 `流量 + 功率`
  - 支持按**品牌筛选 + 型号模糊搜索**来一键带入参数
  - 扩充常见洗车机数据库到 **10 个品牌 / 30 个机型**
  - 新增**功率一览表**可视区：按品牌分组展示所有机型卡片，点击可一键带参到计算器
  - 以 **MJJC 3.0** 作为泡沫壶基准来解读结果
  - 内置常见标准喷芯库（1.10 ~ 2.00 mm）
  - 压力留空时自动估算
  - 工况自洽性提醒
  - 浓泡 / 通用 / 大流量方向建议
  - 支持按**当前已装喷芯 + 想要的泡感**做反推推荐

## 主要功能

### 1. 参数输入
- **最省事的入口**：先按品牌筛选，再搜型号关键词，页面自动填入功率、流量、压力
- 型号搜索不再要求完整匹配，像 `K5`、`UA135`、`雪豹 P5`、`BM1522` 这种简称也能识别
- **流量（L/min）仍然是核心参数**
- **压力（bar）可选**：填了就直接按实测/铭牌压力算
- **功率（W）可选**：
  - 如果压力没填，就拿来自动估压
  - 如果压力也填了，就顺手做功率一致性校验
- **当前喷芯（mm，可选）**：如果你手上已经有喷芯，可以再填一个当前值
- **目标方向**：支持选 `更浓一点 / 更稳更通用 / 更轻松更大过水`

也就是：
- 推荐组合：`品牌筛选 + 型号一键带参`
- 手动推荐：`流量 + 压力`
- 没压力表时：`流量 + 功率`
- 已有喷芯时：`当前喷芯 + 目标方向`
- 最完整：`型号 / 功率 + 流量 + 压力 + 当前喷芯`

### 2. 机型数据库
当前内置了 **10 个品牌 / 30 个机型** 的公开常见参数，主要覆盖：
- 卡赫（6 款）：K2 Classic / K3 Power Control / K4 Power Control / K5 Power Control / K7 Premium Smart Control / K Mini
- 博世（4 款）：EasyAquatak 110 / UniversalAquatak 125 / UniversalAquatak 135 / AdvancedAquatak 140
- 亿力（5 款）：YLQ4430C / YLQ4450G / YLQ4650C / YLQ5680G / YLQ7590G
- 绿田（4 款）：河豚 1400W 款 / 雪豹 P4 / 雪豹 P5 / 逸途 1800W
- 黑猫（2 款）：BM15/11 / BM1522
- 博高（1 款）：BG-K7
- 威克士（2 款）：WU630 / WU633
- 安露（2 款）：AL-K90 / AL-K120
- 拉瓦（2 款）：LV-K3 / LV-K5
- 佳孚（2 款）：JF-1600 / JF-2200

而且每个机型都补了一个简单定位，例如：
- 入门家用
- 均衡家用
- 家用甜点
- 大流量进阶
- 低压大水
- 专业冷水机
- 家用重火力
- 便携锂电
- 进阶家用

说明：
- 这些参数用于**快速初选**，不是官方全量数据库
- 同名机型不同代、不同地区版本，参数可能有差异
- 如果你手上的机器和页面带出来的不一样，优先以铭牌或实测覆盖

### 3. 结果输出
- 合理下限
- 标称推荐
- 合理上限
- 最近标准件
- 推荐区间刻度条
- 已选机型摘要
- MJJC 3.0 甜点带提示
- 已有喷芯该往大还是往小的反推建议

### 4. 标准喷芯推荐
页面会从标准喷芯库里挑出最值得先试的几档，并标注：
- 首选
- 偏浓泡
- 偏通用
- 偏激进
- 偏保守

### 5. 已有喷芯反推
如果用户已经在用某个喷芯，页面现在会根据：
- 当前喷芯大小
- 当前整机工况
- 想要的目标方向（更浓 / 更稳 / 更轻松）

给出这类判断：
- 现在这颗是不是已经接近最佳点
- 先往小一档试，还是往大一档试
- 大概要跨几档，风险是什么

### 6. 风险提示
会提醒以下情况：
- 压力偏低，难出浓泡
- 流量偏小，覆盖效率慢
- 流量偏大，常见小喷芯容易憋压
- 功率、流量、压力三者互相打架
- 当前工况是否偏离 MJJC 3.0 的常见甜点区
- 当前喷芯是否已经明显偏激进或偏保守

### 7. 功率一览表
页面底部新增「功率一览表」可视区，方便浏览全部机型数据：
- 按品牌分组展示所有 30 个机型
- 每张卡片显示：型号 / 功率 / 流量 / 压力 / 定位标签
- 顶部品牌 Tab 可快速筛选某个品牌
- 每张卡片带「算」按钮，点击后自动填入计算器并滚动到顶部触发计算
- 响应式布局，移动端自动切换为单列
- 风格与页面整体深色科技感统一

## 视觉方向

- 冷色荧光高亮
